# twideem.github.io

在部落格和社群的時代，很少有網站繼續提供免費個人網頁的服務，所以用 GitHub Pages 讓我的個人網站繼續下去；雖然這項服務的本意是用來宣傳專案，不是架個人網站用～

感謝 *GitHub*，也感謝一路走來，有 *奇摩*、*PC home Online*、*8d8d*、*Graffiti.net*、*Freewebs*、*Google 雲端硬碟代管網頁*、*Pancake.io* 提供免費又無廣告的網頁空間服務，讓我過足寫網頁文章的癮。

然而，強制 two-factor authentication 的關係，沒手機門號且不想用 Authenticator 自找麻煩的我，決定不再更新與維護這個網站。這個網站，就當我一生寫出來的一部著作，保存在這裡，但願能對找資料的人有點幫助！

既然放棄更新維護，就想說展現一下廢棄後的荒涼感，便將首頁改版為難用得要死的 CLI 模式，必須輸入指令來開啟網頁檔案。這種畫面整片空無沒任何提示不知從何操作起看了就逼退人的首頁，正是我設計的廢棄感 (●'◡'●)

## 如何使用這難用得要死的首頁

首先輸入 ls 會列出可使用的檔案名稱。

然後輸入檔案名稱即可使用檔案。

## 其他功能

clear 可清理畫面。

googler [資料] 可搜尋資料。

lynx [網址] 可瀏覽全球資訊網。
