Windows 8 無法進入遊戲畫面

在 Windows 8 執行《世紀爭霸二》後，遊戲畫面就像自己不斷「最小化」一樣，一直縮小到工作列。

這是 explorer.exe 排程起衝突的問題，只能把 explorer.exe 這個處理程序先刪掉，再進入遊戲。退出遊戲後，再啟動 explorer.exe 來恢復 Windows 桌面管理程序。

整個動作建議寫在批次檔來執行，用純文字文件輸入如下指令：

TASKKILL /F /IM explorer.exe
EE2.exe
START explorer.exe
儲存為 EE2.bat 後，放在安裝《世紀爭霸二》的資料夾裡面。以後都改用 EE2.bat 來執行遊戲。

資料片也是同樣的道理，重新用空白的純文字文件，輸入如下指令：

TASKKILL /F /IM explorer.exe
EE2X.exe
START explorer.exe
然後儲存為 EE2X.bat。
