第一章

晚上，到イムル和監牢的人講話，他會說自己叫アレクス，然後告知フレア…

轉換場景時，會重新計算ちからのたね或いのちのきのみ的數值，其它種子也是。

一到四章，角色身上的道具到第五章會保留下來（任務物品除外），所以拿到はじゃのつるぎ不要把てつのやり賣掉，破關前將身上的錢全部買昂貴裝備放著，將來可以賣錢。

第二章

和國王、ブライ、クリフト、城門的兩個士兵對話完，アリーナ房間的牆壁就會補好。

先向フレノール的神父紀錄，再讀取冒險書，他會站在教會，不再擋住農田。

和小孩對話，得知夜間在墓場拿おうごんのうでわ交換人質的事，對方才會出現。

到さばくのバザー和士兵對話，再回サントハイム探視國王。城後的老人說マローニ治療過發不出聲音的病，到サラン的教會可以找到他，說是用さえずりのみつ治好的。晚間到さばくのバザー和老闆們聊天，得知さえずりの塔可以找到さえずりのみつ…

第三章

救出ドン・ガアデ後，要和ボンモール的國王對話，橋才會修好。

ブランカへの洞窟的老人說要 60000G 開通隧道。有店以後，找エンドール國王商量，他願意用 60000G 買 6 把はがねのつるぎ（ボンモール有賣）和 6 具てつのよろい（エンドール有賣）。不斷買昂貴的武器在櫃台前寄放給妻子賣，再到櫃台後跟妻子說要睡覺，一日一日地賺進大筆金錢，買來交給城內東北角的士兵。

把錢交給ブランカへの洞窟的老人後，出去再回來即開始動工，這時和老人對話，他說開通後會到你店裡通知。回エンドール先到賭場逛一下，再回店問妻子，就會告知開通的消息。

第四章

兩姊妹靠呪文吃飯，偏偏習得呪文的等級不固定，等級到了有可能沒學到想要的呪文。所以不用特別練ミネア到 Lv.13 學ベホイミ，Lv.11 左右就可以打 BOSS 了。

第五章

シンシア躺過的地方藏有はねぼうし。

勇者進入ブランカへの洞窟，經過駐守的士兵前，有機會獲贈 2000 枚賭場遊戲幣。

ミントス東方的祠堂，背道藏有ちいさなメダル。

アリーナ加入後，可用とうぞくのカギ開啟バトランド的寶庫。拿到まほうのカギ後，回エンドール、サラン開門搜刮。

ハバリア北方的小島藏有ちいさなメダル。

キングレオ的 BOSS 有 950 HP，要打很久，有怎麼打都不會死的感覺，不好對付。比起來高等小怪好打多了，所以先往外逛一圈蒐集防具：先到スタンシアラ買ドラゴンメイル、まほうのほうい，但りりょくのつえ不用急著買。順便解任務拿てんくうのかぶと，以及搜刮ちいさなメダル。接著到バトランド開寶箱，就有 15 枚ちいさなメダル到メダル王の城領取ちからのゆびわ。エンドール有第二間防具屋在賣やいばのよろい（海鳴りの祠有てんくうのよろい但怪物太強）。ロザリーヒル買ドラゴンキラー。陣容方面，由於アリーナ還在拿てつのツメ，攻擊力不出色，所以建議陣容為勇者（普通攻擊和 Lv.5 ベホイミ）、ミネア（Lv.13 ベホイミ）、マーニャ（Lv.19 メラミ）、クリフト（Lv.14 スクルト）。

全員集結後，正統的攻略流程為：海辺の村 → スタンシアラ → モンバーバラ → スタンシアラ → サントハイム → バトランド → イムル → ガーデンブルグ → ガーバコタのアジト → ガーデンブルグ → ロザリーヒル → 王家の墓 → 滝の流れる洞窟 → メダル王の城 → 海鳴りの祠 → リバーサイド → 魔神像 → デスパレス → アッテムト鉱山 → リバーサイド → イムル → ロザリーヒル → エルフの里 → 世界樹 → ゴッドサイド → 角笛の祠 → 天空への塔 → 天空城 → 闇の洞窟 → 希望の祠 → 南東アンドレアルの祠 → 南西ギガデーモンの祠 → 北西ヘルバトラの祠ー → 北東エビルプリーストの祠 → デスキャッスル。先前為了順利打倒キングレオ，有些地方事先去過了～

サントハイム的バルザック幾乎都是呪文攻擊，沒必要施展スクルト，所以用耐打且能裝備強力武器的ライアン替換クリフト。

拿到さいごのカギ後，回ブランカ、ハバリア、ボンモール、レイクナバ、コナンベリー、イムル開門搜刮。フレノール南の洞窟有隱藏入口，裡面有いかづちのつえ。

在デスパレス用へんげのつえ變成怪物潛入城內，然後坐在老虎旁邊的座位展開會議。

通過デスキャッスル後，記得用バロンのつのぶえ召喚馬車。

第六章

幕後操弄一切的真正禍源，其實是エビルプリースト！他為了讓ピサロ消滅世間，甚至策劃殺害ロザリー，讓ピサロ憎恨人類…

走完第五章的破關畫面後，系統詢問是否存檔。讀取進度，時間回到決戰デスピサロ前夕，但這次不要前往デスキャッスル打倒他，改而讓ロザリー復活，化解ピサロ的憎恨。

前往ロザリーヒル和エルフの里探聽情報，得知せかいじゅのはな可以讓ロザリー復活。到ゴットサイド的祭壇，進入謎のダンジョン，擊敗エッグラ與チキーラ，繪畫中的世界樹會綻放せかいじゅのはな。前往摘取，在ロザリー墓前讓她復活後，帶去見ピサロ。最後一起前往デスパレス打倒エビルプリースト！

可能遊戲設計上資料結構只配置 9 人份，所以ドラン會離開隊伍，讓ピサロ加入。

ピサロ不受詛咒裝備的影響，且呪文學完至少要 Lv.60。
