七龍珠 3 悟空傳

操作與玩法

指令選單

按 SELECT 可叫出指令選單。

屬性

升級時要分配屬性點：

スピード　　　速度，提升對雜魚的攻擊力和防禦力。
パワー　　　　力量，提升對頭目的攻擊力。
テクニック　　技術，提升必殺技的傷害。
タフネス　　　強韌，提升對頭目的防禦力。
我都配：速 2 力 1 技 1 強 1。

有些 BOSS 的強度，是根據玩家的屬性計算出來的，所以不見得屬性越高越好，到時候 BOSS 反而更強更難打。

目前等級練到 5 就上限了，會有很長一段時間打怪沒修行值，隨著劇情推進會提升等級上限。

因為芭蕉扇弄丟了，所以龜仙人會加入。然後回到 2 的地方，龜仙人施展龜派氣功撲滅火焰。然後按：

はなす → ぎゅうまおう → いどう → もどる

4

去找琪琪，按：

はなす → ぎゅうまおう → いどう → もどる

獲得飛行器，用法是按 SELECT 跳出指令選單，選：カード → アイテムカード → ひこうき，即可使用它快速前往地圖中標示紅點的位置，只能用一次。

3

再回龜仙屋，可用飛行器直接過去，然後按：

いどう → カメハウス → はなす → ブルマ → いどう → パンティさがし

接著往左下方走，選一張牌獲得內褲，有白色粉紅蝴蝶結和灰色黑蝴蝶結兩種，隨機取得。

回原來的位置，按：

はなす → かめせんにん → いどう → もどる

拿到ギャルのパンティ（白色粉紅蝴蝶結）會獲得筋斗雲，ブーギャルのパンティ（灰色黑蝴蝶結）則是飛天龜。用法是按 SELECT 跳出指令選單，選：カード → アイテムカード → キントウン 或 コガメラ，即可使用它快速前往地圖中標示紅點的位置。筋斗雲會消耗 10 點能量，飛天龜會損失 5 點生命，由於能量不像生命容易恢復，所以飛天龜反而比較好。

飛天龜以高速 360 度旋轉的方式飛起來，坐在上面的龜仙人一下來就被轉得頭暈眼花，因此遊戲才會設定扣生命吧？

修改

VirtuaNES

#1 0405-01-FF       生命 255
#1 0406-01-64       能量 100
#0 042D-02-3606     第一張牌「必」且數值 7
#0 0407-03-0F423F   修行值 999999
複製貼上到純文字文件，然後以 *.vct 副檔名儲存，在 VirtuaNES 的 Tool - Cheat code - Load 載入使用。

Nestopia

<?xml version="1.0" encoding="UTF-8"?>
<cheats version="1.0">
    <cheat enabled="1">
        <address>0x0405</address>
        <value>0xFF</value>
        <description>生命 255</description>
    </cheat>
    <cheat enabled="1">
        <address>0x0406</address>
        <value>0x64</value>
        <description>能量 100</description>
    </cheat>
</cheats>
複製貼上到純文字文件，然後以 *.xml 副檔名儲存，在 Nestopia 的 Options - Cheats - Permanent Codes 按 Import.. 載入。
