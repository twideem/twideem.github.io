修改

預設只開啟玩家不死，其它功能請依自己需求勾選。

VirtuaNES

#1 04C2-02-FFFF     1P 2P 不死
#0 04C4-02-0000     3P 4P 瀕死
複製貼上到純文字文件，然後以 *.vct 副檔名儲存，在 VirtuaNES 的 Tool - Cheat code - Load 載入使用。

Nestopia

<?xml version="1.0" encoding="UTF-8"?>
<cheats version="1.0">
    <cheat enabled="1">
        <address>0x04C2</address>
        <value>0xFF</value>
        <description>1P 不死</description>
    </cheat>
    <cheat enabled="1">
        <address>0x04C3</address>
        <value>0xFF</value>
        <description>2P 不死</description>
    </cheat>
    <cheat enabled="0">
        <address>0x04C4</address>
        <value>0x00</value>
        <description>3P 瀕死</description>
    </cheat>
    <cheat enabled="0">
        <address>0x04C5</address>
        <value>0x00</value>
        <description>4P 瀕死</description>
    </cheat>
</cheats>
複製貼上到純文字文件，然後以 *.xml 副檔名儲存，在 Nestopia 的 Options - Cheats - Permanent Codes 按 Import.. 載入。
