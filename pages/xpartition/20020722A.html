DOSBox

熱鍵

Ctrl + F11減少 CPU speed cycles。
Ctrl + F12增加 CPU speed cycles。
Ctrl + F7減少 Frameskip。
Ctrl + F8增加 Frameskip。
Ctrl + F10解除 DOSBox 對滑鼠游標的鎖定。
Ctrl + F5抓取畫面。
Ctrl + Alt + F5錄製畫面。
Ctrl + F6以 WAVE 格式錄音。
Ctrl + ALT + F8以 MIDI 格式錄音。
Ctrl + F9關閉 DOSBox。
Alt + Enter切換視窗或全螢幕。

以百分比控制 CPU 速度

cycles=max

驅動光碟機

mount D "D:\"

驅動含音軌光碟機

mount D "D:\" -usecd 0 -ioctl

把遊戲掛載成光碟機來執行

mount D "C:\GAME" -t cdrom
不要驅動實體光碟機，兩者不能同時存在。

掛載映像檔

imgmount E "IMAGE.ISO" -t iso

可將多個映像檔掛在一起，例如：

imgmount E "DISC_1.ISO" "DISC_2.ISO" -t iso

遊戲提示要換片時，按 Ctrl + F4 切換 DISC_1.ISO 和 DISC_2.ISO。

CD 音軌

關於映像檔，ISO 是早期的格式，優點是蔚為標準，缺點是無法包含光碟音軌。有些以 Alcohol 120% 轉為 MDF/MDS 格式的遊戲光碟會含音軌，轉為 ISO 的話就沒有 CD 音源。但 DOSBox 不支援 MDF/MDS 格式，這時我們可以找一些程式工具，轉為 DOSBox 有支援的 BIN/CUE 掛載使用。

掛載 CUE 檔的話，BIN 檔名不要亂改，因為 CUE 裡面有 BIN 檔名資訊，會辨識不到。

簡化啟動程式介面

在捷徑後面加入 -noconsole，例如：

"C:\Program Files (x86)\DOSBox-0.74\DOSBox.exe" -noconsole
