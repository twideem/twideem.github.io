Rockman X4

第一次雙主角，帥氣到不行的 ZERO 擄獲廣大玩家的心！加上為了讓更多人接觸這系列作品，難度設計得容易許多，是最多人視為經典的洛克人！

攻略順序

X

WEB SPIDER → SPLIT MUSHROOM → CYBER PEACOCK → STORM OWL → MAGMA DRAGOON → FROST WALRUS → JET STINGRAY → SLASH BEAST

Zero

MAGMA DRAGOON → WEB SPIDER → SPLIT MUSHROOM → CYBER PEACOCK → STORM OWL → FROST WALRUS → JET STINGRAY → SLASH BEAST

密技

X

1、選人物時把框移到洛克人上。
2、按 ╳ 兩下。
3、按 ← 六下。
4、按 L1 + R2 不放。
5、按 ○ 開始遊戲。
6、成功的話，主角身上會有紫色部位，但還未真正獲得終極鎧甲，先到 WEB SPIDER 拿到一個部位才算完成。

Zero

1、選人物時把框移到 Zero 上。
2、按住 R1 不放。
3、按 → 六下。
4、放開 R1。
5、按住 ╳ 不放。
6、按 ○ 開始遊戲。
7、成功的話，Zero 會是黑色的。

Street Fighter X Mega Man

簡介

CAPCOM 紀念洛克人25 周年，與正在製作快打旋風洛克人同人遊戲的 Seow Zong Hui 合作，推出 Street Fighter X Mega Man，並免費放在官網下載：

http://www.capcom-unity.com/mega_man?no_redir=1

攻略順序

Ryu
Chun-Li
Crimson
Dhalsim
Blanka
Rose
Rolento
Urien

密技

在標題畫面，長按 SHOOT 和 JUMP，聽到特殊音效，再進入遊戲，會開放「星影圓舞」的能力！不但可以按 ↓↘→＋SHOOT 打「波動拳」，更可以同時按 SHOOT 和 SWAP1 進入「星影圓舞」狀態，攻擊、速度、跳躍能力通通上昇。
