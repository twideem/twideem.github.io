世紀爭霸

還記得當時在店裡看到這款遊戲，盒上寫著：

橫跨人類歷史 50 萬年，從舊石器時代、新石器時代、文藝復興時代、工業時代到第一次及第二次世界大戰，再跨越到未來的微子時代…

我巴望這樣一款遊戲好久了，驚呼終於等到！一秒鐘猶豫的時間都沒有，馬上買一盒回家…你知道的，畢竟電腦遊戲不是百元有找的便宜東西，在店裡總是這盒拿拿、那盒搖搖，反覆比較該買哪一款遊戲，猶豫再三。

雖然現在的眼光來看，這遊戲在內容上其實還有進步的空間，但畢竟是當年唯一能從遠古時期玩到太空時期的即時戰略遊戲，所以也沒想那麼多，快樂地沉迷十數個月的時光～

感想

承前言…

要說缺點嘛，就是「戰役遊戲」的娛樂性不高，不是不能建造某些建築物，就是不能生產某些單位，不能玩得盡興。所以我都玩「隨機地圖」，但電腦嚴重作弊，它根本不用耕田挖礦就能生產軍事單位、推進時代，所以攻擊村民想讓對方進展得比我方慢是沒用的[1]。

因此這只能是一款用軍事征服對手的遊戲，不能想像自己在經營一個國家，不打算侵略對方，想打造兩國共存的美好世界，因為這樣做的下場，就是好不容易發展到工業時代的你，被已經原子時代的電腦給滅掉 XDDD

但說這是缺點也蠻過份的，因為開發者原本就是想設計一款軍事征服的遊戲，不打算要你把時間浪費在發展文明上。想「經營國家」的話，應該玩「文明帝國」系列，但那是回合制的戰略遊戲，沒有即時戰略遊戲有臨場感，所以我依然發揮「腦補」的能力，把「世紀爭霸」當「文明帝國」玩，而「快樂地沉迷十數個月的時光～」直到《世紀爭霸２》推出。

其他

Rick Goodman 是《Age of Empires》首席設計師，他為了設計囊括人類所有歷史的即時戰略遊戲，不惜離開共同創辦的 Ensemble Studios，另外成立 Stainless Steel Studios 開發了《Empire Earth》，果然大受好評，成為系列作的開端。

不過，Rick Goodman 操刀只有一代，往後的《Empire Earth II》《Empire Earth II: The Art of Supremacy》《Empire Earth III》其實是 Mad Doc Software 設計的，甚至一代資料片《Empire Earth: The Art of Conquest》都是如此。

是否有什麼內情不得而知，但一代其實是 Stainless Steel Studios 和 Mad Doc Software 共同開發，由 Mad Doc Software 延續 EE 系列還說得過去。

Stainless Steel Studios 接下來的作品不是《Empire Earth II》，而是《Empires: Dawn of the Modern World》，一款節奏較快，適合連線對戰的即時戰略遊戲。

雖然不是 Rick Goodman 所設計，Mad Doc Software 的《Empire Earth II》其實評價相當不錯：令人激賞的 3D 繪圖畫面，像是氣候系統。更適切的軍事單位，尤其每個民族都有獨自的單位，不會看到俄羅斯生產 F-15 和 B-29。加上引進外交與貿易系統，除了戰爭之餘更能體驗發展文明的樂趣，可說更進一步強化 EE 系列的深度。

然而 Mad Doc Software 卻在《Empire Earth III》砸了 EE 系列的招牌，民族縮減成：Western（西方）、Middle Eastern（中東）、Far Eastern（遠東）三種文明，時代也縮減成 Ancient（古代）、Medieval（中世紀）、Colonial（殖民）、Modern（現代）、Future（未來）五個，也沒有劇情戰役，只有一個 World Domination（征服世界）遊戲模式。結果以史詩般壯闊聞名的 EE 系列，變成一款玩起來很乏味又空洞的遊戲。罵聲四起，連 Mad Doc Software 都從自己的產品名單上移除《Empire Earth III》，表示這確實是一款失敗的、不該拿出來賣的作品。

但《Empire Earth III》卻展示出 Mad Doc Software 的 3D 繪圖遊戲畫面是多麼精美！所以我並未對這家公司失望，在這 PC Game 完全被次世代主機打趴的年代，如果願意繼續努力嘗試，難保不會出現讓電腦玩家大呼過癮的即時戰略遊戲啊～

[1] 除非讓電腦連一隻村民都沒有，無法建築城鎮中心，就無法推進時代。
