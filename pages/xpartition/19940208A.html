軒轅劍貳

修改物品欄

遊戲一開始，物品欄空空如也，正好適合用 Cheat Engine 找出第一欄位置。

進安定村，找雜貨攤販，按順序買：斗笠、草鞋。

在 Value Type 選擇 Array of byte，然後輸入 2a 01 11 01，就能找到第一欄的 Address。

在找到的 Address 上，按 Ctrl + B，進入 Memory Viewer，就能按照順序填入如下道具代碼：
