模擬市民典藏年鑑

因為覺得划算，主程式＋美麗人生＋歡樂派對＋非常男女＋歡樂假期＋寵物世代＋超級巨星＋魔法世界，只賣 790 元，所以就買來收藏了～

聽著輕鬆柔和、慵懶甜美、浪漫性感的 Bossa Nova 音樂（維基百科描述得真好），看著螢幕裡市井小民正常地過著市井小民的生活，倒也打發了不少時間。

由於沒什麼養成要素，也就是沒遊戲目標，只是看著人物的多個綠條消漲，所以玩膩後很少再裝來玩。

加上 EA 食髓知味，模擬市民 3 開始，很明顯將遊戲設計成主程式簡陋，好刺激玩家往後買他一連串資料片賺錢，沒買嘛，主程式很兩光，買嘛，資料片又賣得跟遊戲一樣貴，整個系列作變成有錢人家才玩得起的遊戲，開始惡名昭彰，於是連關注都不想關注，當作沒這遊戲。

密技

按 Ctrl + Shift + Alt + C，即可輸入以下密技：

klapaucius = $1000
rosebud = $1000（資料片）
set_hour 1~24 = 時間
set_speed -1000~1000 = 遊戲速度
interests = 個性和興趣
autonomy 1~100 = 想法

! = 重做上一個密技
; = 分開多個密碼指令

需求值全滿

輸入 move_objects on 密技，可以在購買模式把人物丟掉，然後回到生活模式按人物頭像，人物會重新出現，且需求值全滿。
