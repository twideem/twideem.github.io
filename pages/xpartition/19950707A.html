仙劍奇俠傳

《仙劍奇俠傳》是我第一款「中文 RPG」！看懂得角色在講什麼之後，才體會到 RPG 的樂趣是劇情，而不是打怪累積經驗值提升等級[1]。

然而，結局卻讓我心靈受創。為了早日讓逍遙和靈兒在一起，我半夜爬起來偷開電腦努力練功。為了讓逍遙帶給靈兒幸福，我還問同學怎麼用 FPE 修改遊戲，讓角色變強，遲遲不會用的我，問到同學都快厭煩了 :)

誰知道破關後，男女主角天人永隔，反而無法在一起。我心想：「如果不打敗大魔王，靈兒反而可以活著，與逍遙在一起吧？」「我這些日子辛苦練功，只是促成一場悲劇而已。」從此我不太喜歡接觸男歡女愛的故事內容，偏好周星馳或蠟筆小新那種沒水準的搞笑內容。

仙劍奇俠傳 98 柔情篇

下載

這是大宇在中國將《仙劍奇俠傳 Windows 95 版》改為簡體中文後推出的版本，官方於 2011 年開放免費下載。官網把頁面刪除了，但免費下載的檔案還留在官網裡面：

http://update1.baiyou100.com/resource/Pal98rqp.zip

在繁體中文作業系統會顯示亂碼，可搜尋下載繁體中文化、或直接下載網友轉換好的版本解決。繁體中文化改用 sdlpal95.exe 執行遊戲，但在 Windows 10 會有對話文字跳動的缺點，建議繼續用 Pal.exe 執行遊戲。

全螢幕執行

在 Windows 10，Pal.exe 必須以 Windows XP (Service Pack 2) 相容模式執行。建議勾選「在 640 x 480 螢幕解析度下執行」，進入遊戲按 F12 切換為全螢幕，就能在 4:3 尺寸玩，不然 16:9 的仙劍奇俠傳不怎麼原味。

可以在遊戲視窗的標題列上按滑鼠右鍵，點「補丁設置」，設定每次執行遊戲都全螢幕。繁體中文作業系統顯示亂碼看不懂如何設定的話，按最左下角的按鈕，然後修改「PAL.使用者名稱.ini」，改寫 Windowed=0 即可。

視窗執行

Windowed=1 的話，WindowWidth 和 WindowHeight 的實際遊戲視窗會較大，寬度會加上視窗框線大小，高度會加上標題列大小。

以 Windows 10 來說，寬度會加 2，高度會加 27，所以可設為寬 1120、高 700。
