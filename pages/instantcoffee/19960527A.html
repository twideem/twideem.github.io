JDK 無法重新安裝

不正常移除 JDK 後，想重新安裝時，安裝程式表示已安裝過，詢問使用者是否重新安裝，確定後卻無法順利完成移除已安裝程式的動作，導致無法重新安裝。

解決辦法

執行 regedit，依序進入……

HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Installer\Products

底下有許多機碼，一個個點看看，找出 Product Name 為 Java SE Development Kit 開頭的資料，然後將整個機碼刪除。

如此就能重新安裝 JDK，而且不用重新開機。

不行的話，只好一個一個搜尋 Java 來刪除。注意！不要刪到 Microsoft 的 Java VM。
