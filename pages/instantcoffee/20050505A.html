private、protected、public

Java 語言有四種存取權限，用來決定符合哪些條件的 class 可以存取 field（資料欄位）和 method（操作方式）：

1、只有 class 本身能存取。
2、只有同 package 的 class 能存取。
3、使用 extends 而來的 class 就能存取。（即使不同 package）
4、任何 class 都能存取。
攸關存取權限的 modifier（修飾子）有三種：private、protected、public。

private

將 field 與 method 設定為 private 的話，只有自己能存取，其它 class 存取的話，編譯器會警示錯誤。

default

未使用 modifier 的話，Java 會預設使用有別於 public、proteted、private 的存取權限，因此稱為 default。

default 只有同 package 的 class 能存取，底下示範無法執行的例子。

A.java

B.java

main

因為 A 的 f1() 是 default，只限同 package 的類別存取，因此 t.B 無法存取 s.A 的 f1()。

你把 B 的 package 改成 s 看看，就可以通過編譯了。

protected

protected 可將 field 與 method 設定為第三種存取權限。

A.java

B.java

main

雖然 t.B 與 s.A 是不同 package，但因為 B 繼承 A，因此 B 可以存取 A 的 f1()。

public

將 field 與 method 設定為 public 的話，任何外部 class 都能存取。
