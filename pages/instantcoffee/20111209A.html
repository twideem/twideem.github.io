Path 介面與 File 類別的相容性

Paths.get() 傳回的是 java.nio.file.Path 介面，可以做比 java.io.File 類別更多事情，使用起來卻更簡單，未來 Path 介面很可能漸漸取代 File 成為檔案的主流。

屆時遇到新舊功能的相容問題，可使用 Path.toFile() 和 File.toPath() 互相轉換。
