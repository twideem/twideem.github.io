String==String？

由於 Java 的文字資料都是 String 物件，因此當你用 == 來比較文字資料是否相等時，你比對的其實是兩個 String 物件所指向的記憶體位址是否相同，而不是兩筆資料是否相等：「雖然執行結果其實是你想要的，看起來很正常。」
