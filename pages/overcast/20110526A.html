Linux Mint 的崛起

自從 Ubuntu 預設讓人用了會幹聲連連的 Unity 使用者介面，大家紛紛轉投基於 Ubuntu 而來、繼續使用 GNOME 桌面環境的 Linux Mint，這就是 Linux Mint 崛起的原因！

由於 Linux Mint 才是大家想要的 Ubuntu，現在成為最受歡迎的 Linux 發行版！

沒想到 GNOME 3 也走 Unity 路線，而且主打精簡介面的操作風格，精簡到你跟本不知道怎麼按、甚至不知道在操作哪一個程序[1]。如果預設這種使用者介面，幹聲連連的程度保證超過 Unity！於是基於 Gnome 的 Linux Mint 分為下面兩種版本：

Cinnamon：改造 GNOME 3 成適合 Linux Mint 的使用者介面。（由 Linux Mint 開發）
MATE：繼續整合已停止維護的 GNOME 2 桌面環境。（由 Stefano Karapetsas 開發）
但好不好用見仁見智，幾度在 KDE、GNOME、Xfce 跳來跳去的 Linus Torvalds，就很欣賞 GNOME 3 而使用它…雖然對他來說這是原則或品味的問題，不是好不好用的問題。

[1] 這是剛推出時的情況。隨著版本演進，GNOME 3 不斷根據使用者的建議修正設計，現在已經沒難用地那麼誇張。
