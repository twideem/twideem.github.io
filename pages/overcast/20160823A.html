其它

其他的就下載 Universal USB Installer 將 Live CD 製作成 Live USB 試試看吧～

Live CD 只能藉由開機時驅動光碟機或 USB 隨身碟，藉由安裝程式進入作業系統，無法寫入資料保存設定檔，因此每次開機就重新來過。而 Live USB 不一樣，可以保存設定檔，因此就像把作業系統安裝在隨身碟，你可以修改中文輸入法、更改桌布、安裝軟體，開機後就跟先前一樣。

不過也有像 elementary OS 這種本身就夠用的 Live CD！除了可切換中文介面，還內建中文輸入，只要開啟終端機輸入 ibus-setup，再加入新酷音即可！網路自動 IP 連線，打開瀏覽器看 YouTube 影片都沒問題。由於 Live CD 載入速度比 Live USB 快，對不求把隨身碟當真正作業系統用，只想四處快速開機上網打打資料的人，桌面設計得美輪美奐的 elementary OS 真是四處拿來羨煞人的好物！

但 Live USB 可以直接進入桌面，Live CD 會先停留在安裝畫面等按下一步，所以雖然 Live CD 不用跑一堆設定組態，載入速度較快，但整體上依然是射後不理的 Live USB 來得方便 XDDD
