何以為 Linux 痴狂

承續《家用電腦的時代看 Linux》，我聲稱在家用電腦的時代，Linux 比不過 Windows，但目的不是在貶低 Linux，而是說出 Linux 在競爭 Windows 版圖上落後的事實。

正視問題，才能尋求 Linux 該如何讓人痴狂的答案。

Linux 能夠讓使用者打造出設備系統、而不只是作業系統

Linux 雖然不適合做為家用電腦的作業系統，卻有很多人為之痴狂，這些人痴狂的原因值得一提：「Linux kernel 是可以修改的，能讓人客製出迎合個體需求的作業系統，或者說透過 Linux 可以讓人改造出適合自己的電腦設備。」這才是 Linux 早先獲得駭客們青睞的關鍵因素，而不是讓電腦使用者變得幼稚的桌面環境。

因此 Linux 可以不用在家用電腦的觀點上與 Windows 競爭，如果往「修改內核」進行人性化研究，Linux 便能突破家用電腦的時代，創造新的電腦時代遠遠把 Windows 拋在後頭[1]。

只是 Linux 的 kernel 依然屬於駭客的領域，在家用電腦時代，懶人們不可能對那種東西有興趣。我們想要電腦做什麼事，就是上網找看看有沒有軟體可以 download，不僅沒有意識到整個電腦設備的功用其實被作業系統限制住了，也從未被告知其實自己可以打造電腦設備的主要用途。如果 Linux 能回歸「內核」的初衷，然後想辦法讓電腦懶人也能隨心所欲修改系統核心享受打造電腦設備的感覺，這天底下還有誰會是你的對手？

當舊電腦跑不動新版 Windows 時，我們會認為這台電腦沒用該換新的了。但如果你會 Linux kernel，你可將這台電腦打造成其他用途的設備，不只是用來跑桌面而已。這才是 Linux 讓人痴狂的魅力所在。

回歸電腦的樂趣

我和許多人一樣，每次裝完新的 Linux 發行版，看到變得更精美、更實用，就可以興奮兩三天；那幾天對 Linux 是愛不釋手。

但比起 Windows 而言，這就是我想要的 Linux 嗎？

70 年代，電腦的樂趣在於如何為這套設備打造它的功能、定位它的用途，以顯現這套設備該有的價值。90 年代，電腦的樂趣變成如何在作業環境底下運行想要的動作而已。

或許「設備」太「機器」了，討論作業環境底下運行的動作還比較「資訊」一點～

[1] 能修改系統核心重新打造出不一樣的作業系統，那就不叫 Windows 了。若 Microsoft 真走到這地步，也會以另一種商品型態出現才對。所以 Windows 必敗無疑……如果 Linux 真的能做到除了駭客以外，連一般使用者也能玩系統核心的話。
[*] 本文所指的年代，是指 20 世紀。
