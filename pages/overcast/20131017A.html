推廣 Linux 二三事

一！

在台灣幾個主要 Linux 論壇經常看到這樣的情形…

當有人抱怨 Linux 不夠友善時，就會有人指出問題在你不肯學習，所以不會用，你只會用 Windows。

但之前我們推廣 Linux 給他們用時，說法明明是：「Windows 能做的事，Linux 通通做得到，而且做得更好。」

先騙說會用 Windows 就可以輕鬆學會 Linux，事後卻怪對方不努力學習 Linux～

二…

說真的，別一副「包裝僅供參考，與實際內容物無關」的態度，欺騙與傷害使用者的感情。

1998 年推出的 Windows 98 做得比 2000 年的 Red Hat Linux 6.1 + CLE v0.9 好。2001 年推出的 Windows XP 做得比 2002 年的 Mandrake Linux 8.2 好，甚至也比 2006 年的 Ubuntu 6.10 好。到了 2009 年 Windows 7 還是做得比 2010 年的 Linux Mint 8 好。

各個 Linux 發行版雖然不至於「便宜沒好貨」，但優勢在於「俗擱大碗」，從 Windows 改用 Linux 就是賺到！然而標榜「Windows 能做的事，Linux 通通做得到，而且做得更好。」是有問題的。

從家用或辦公用的需求來看，Windows 可以做的事，通常 Linux 不會做得比較好，反而 Windows 做得到的事，Linux 做不到的情況比較多。然而，對電腦工程師來講，他是拿 Linux 跟 Windows Server 比，這時 Linux 做得到的事比 Windows 更多。

電腦對工程師來說是「計算機」，計算研究數據用的，而不是「家電」。但既然要拿 Linux 跟 Windows 比，應該用「家電」或者「辦公室器材」的角度才對，因為這才是對方主打的版本與強項～

我想這是「包裝與內容物不符時僅供參考」的問題所在。如果我們要說服使用者 Linux 比 Windows 更棒，應該改變立場，從應用 Windows Server 的情形，突顯 Linux 是效率更好的作業系統！

三？

Windows 是「設計給消費者使用的作業系統」，而 Linux 是「設計給使用者消費的作業系統」，光價值觀就有差異。

「要掏錢買的 Windows 不管怎麼設計，我們永遠嫌他做得不夠好。」
「免費的 Linux 不斷在進步，我們一直享受陪伴他成長的過程。」
雖然 Linux 逐步取代 Windows 用途的過程進展得很慢，但在 Windows 取代不了 Linux 用途的基礎下，選擇 Linux 是正確的選擇！

我們不該再以 Windows 為訴求，連哄帶騙邀請使用者加入 Linux 的行列，而是告訴他們使用 Linux 電腦有哪些樂趣：「電腦不只是拿來上網、聊天、打字、玩 GAME 用的，而是可以拿來學習一技之長用的！」

不過，在這平板電腦蔚為主流的行動上網時代，學習電腦似乎是種退流行的想法了！但 MS-DOS 與 Windows 時代想學電腦的動機，與在 Linux 學習電腦有著不一樣的層次。以前是為了職場需要，而學 MS-DOS 電腦該怎麼操作、學 Windows 軟體該怎麼用，但 Linux 是為了發揮電腦更多用途，而吸引我們想探究、學習它，這真的是一種樂趣，而不只是動機！

因此，我開始自省該如何以 Linux 本身為訴求，推廣大家使用它，一起享受學習電腦的樂趣～
