文字編輯器（editor）相關術語

abbreviation, code template, snippet

樣板；簡單打幾個字，就能輸出完整語法結構。如果沒錯的話，這應該是由 Zen Coding 套件帶動的潮流。

auto completion, code insight

撰寫原始碼時，文字編輯器自動判斷接下來可能的語法有哪些，列出清單以供選用。又稱。

files manager

檔案管理視窗。

fold

摺疊；可以折疊某區塊的程式碼和語法，通常位於行號右邊。

highlight

高亮度；以不同顏色，顯示不同文字要素。

hint

即時檢驗語法，不通過的話所給予的提示。

indent

縮排。

insight

根據使用者輸入的字詞，即時列出清單，以供使用者候選剩下的字。

mark occurrences

現出標記；在某變數名稱點一下，所有變數名稱都會標示起來。

matching

匹配；自動標示另一個與其成雙成對的元素。

navigator, treeview

結構分析器；將原始碼依結構剖析成樹狀結構。

object inspector

將物件的屬性、方法、事件，以可下拉選擇的表格整理出來，方便選用。

projects

專案管理視窗。

scheme, style, theme

主題；可切換成其他不同配色的高亮度文字。

snippet, template

程式碼片段、樣板；只打一小段關鍵字，按 Tab 鍵轉為完整內容。

trim trailing

消除每行後面的空白字元。
