我的 Linux 歷程

我用過的 Linux 發行版

我念書時家裡沒網路[1]，而且不是資訊科的，學校不可能提供我下載 Linux 安裝光碟的環境，所以都靠買書獲取附贈的 Linux 作業系統安裝光碟。

其實這樣蠻不錯的，這種書比較像雜誌，不像磚頭般的電腦圖書那麼貴，當作花 299 元買一套含有說明書的 Linux 作業系統，還是很划算！[2]

所以我的 Linux 歷程，就用買過的這些書名來回顧吧～

陳俊宏《Red Hat Linux 7.2 + CLE 1.0 系統安裝 DIY》
施威銘《Linux-Mandrake 8.1 玩家實戰》
施威銘《Red Hat Linux 7.3 實務應用》
施威銘《Mandrake Linux 9.2 魔法護照》
MIS 2000Lab《Mandrake Linux 10 學習真簡單》
施威銘《Mandriva Linux LE2005》
施威銘《Fedora Core 4 Linux 保證會》
離開學校開始工作賺錢，先後裝了 Cable 和 ADSL 寬頻，但還是習慣買書安裝 Linux，因為性急的我等不了用 512K 下載三到四片 650 MB 光碟映像檔燒成 CD 片～

大概一年多後，我推薦使用的 Mandriva Linux，公司居然將創辦人 Gaël Duval 炒魷魚，而且沒了 Gaël Duval 的 Mandriva 再也不是 Mandrake！加上這時工作不順遂的關係，就沒心情碰 Linux 了！

數年後，偶爾會用 VirtualBox 裝 Ubuntu 來玩，直到 11.04 版預裝既礙眼又難用的 Unity 介面，連 Ubuntu 也不想碰。但也因此見識 Linux Mint 精美耐用的魔力，覺得這是 Linux 發行版最極致的表現了！

始終用虛擬機器跑並不過癮，所以用 USB 隨身碟開機方式執行的 Slax，讓我重新找回 Linux 的感動！

隨後在一台打算報廢的舊筆電裝 Lubuntu 13.10 做為 Linux 平台專用機。

經歷 Lubuntu 18.10 和 Lubuntu 19.04，覺得 LXQt 沒有過去 LXDE 美好，像原本憨厚呆萌的小孩長大了，開始為了吸引眾人注意而調皮搗蛋起來，於是改用氣質一點的 elementary OS。

關於 CLE

真的很感謝黃志偉先生率成員們努力推行 CLE（Chinese GNU/Linux Extensions；Linux 中文延伸套件計劃），讓台灣人不僅都能順利安裝 Linux，還能真正的使用它來辦公。

以前，玩 Linux 的人最大樂趣並不是使用它，而是成功在機器上安裝它。

如果沒有 CLE 的熱心貢獻精神，當年我根本沒有能力投入 Linux 領域。

關於 Linux-Mandrake

在大家搶著學習用 Red Hat Linux 架站時，我鍾愛的是有華麗桌面、搶鮮引用各種最新應用軟體的 Linux-Mandrake。重點是安裝速度很快，我懷疑它直接把光碟片的資料複製到硬碟就灌好了！

在我接觸 Linux 沒多久的 2002 年，Microsoft 大執法，正式取締校園盜版使用 Windows 和 Office 的行為。如果你覺得七年級生對微軟很反感，就是這麼來的，當年我們就是這群用盜版軟體準備被取締的學生 XDDD

Linux-Mandrake 成為我努力推薦給大家、希望用來取代 Windows 的解決方案！

當年推廣 Mandrake，注入了熱血與青春、充滿了前景與希望，所以後來使用 Slax 和 Lubuntu，也很難取代 Mandrake 在我心中的地位。

但 Mandrake Linux 的優點，後來成為 Mandriva Linux 的缺點。只顧搶鮮引進最新的軟體，卻很少回頭整理與優化既有的功能，結果版本越新，越覺得系統有挖東牆補西牆的破爛感。

這點來講，或許正是 Ubuntu 成功的地方吧？它只加入經過調校、且桌面使用者真正會用到的東西，然後獨創一套穩健的組件管理機制，透過網路由使用者自己決定想要下載的東西。

在我與 Gaël Duval「共進退」拒用 Mandriva Linux 後，沒多久 Ubuntu 取代了 Mandriva Linux 在桌面作業系統的地位，成為最多人使用的 Linux 發行版。

不過，我並未跟著使用 Ubuntu，前文說過是工作不如意的關係沒心情再碰電腦，其實它幹掉我最喜歡的 Mandrake 才是主因吧 XDDD

關於 Lubuntu

因為我的 Acer Aspire 4736ZG 筆電硬體出現瑕疵，跑不動 Windows 7，但改裝 Lubuntu 13.10 就又活跳跳，於是就這樣做為 Linux 專用機。

雖然有堪稱 Mandriva Linux 接班人的 Mageia，但我並不知道有這個發行版 （艸）

即使知道了，我還是比較喜歡輕量級、簡單就好的 Lubuntu。

等於跟著使用 Ubuntu 系列，拋棄了 Mandrake～

關於 elementary OS

雖然沒有 Linux Mint 耐操，但 elementary OS 已經非常穩健、基本上操不壞了！加上 elementary OS 的操作介面明明更美觀，卻不像 Linux Mint 笨重，而像 Lubuntu 一樣走輕量路線，所以放棄 Lubuntu 19.04 後，決定選擇 elementary OS，不是 Linux Mint。
