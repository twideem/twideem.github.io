我愛小企鵝輸入法

Fctix 是 Free Chinese Input Toy for X 的意思。

特色

我偏好使用 Fctix 的新酷音，它可以按 R-Shift 切換輸入法，跟我在中文輸入法難用到爆的 Windows 8 最後折衷做法是固定開啟新注音輸入法只靠 R-Shift 中英切換一樣～

除此之外，Fctix 與 Lubuntu 的軟體相容性很好，不會有其它輸入法候選字欄位置跑掉、或者功能異常導致當機的問題。

安裝

apt-get install fcitx fcitx-chewing

心得

全形符號切換

預設是「Ctrl + ,」，雖然我想自訂為「Shift + Space」，但沒有效果，不曉得這組合鍵被哪個程序給佔用了 Orz

虛擬鍵盤

預設是「Ctrl + Shift + B」，我另外自訂為「Ctrl + .」，但微軟新注音是「Ctrl + Alt + ,」。

如何輸出全形英文和數字？

先切換到全形符號，再叫出虛擬鍵盤敲英文和數字。

只按 R-Shift 鍵切換中英輸入方案

進入設定輸入法的介面，在「輸入法」頁籤中，清單要有如下兩項，且照順序排列：

新酷音
鍵盤 - 英語 (美式)
在「全域設定」頁籤中，勾選「顯示進階選項」，然後在「快捷鍵」頁籤設定：

額外的啟用輸入法快捷鍵 = 右Shift
切換虛擬鍵盤 = Ctrl+.
切換全形字元模式 = Ctrl+,
在「程式」頁籤設定：

預設輸入法狀態 = 作用
在「輸出」頁籤設定：

候選字數 = 8
未聚焦視窗時，不要提交預編輯文字 = 勾選
（其它取消勾選）
在「外觀」頁籤設定：

只有預編輯字串時不顯示輸入視窗 = 勾選
（其它取消勾選）

關閉用不到的項目

進入「附加元件」→「進階」，然後取消勾選如下項目：

快速輸入
Freedesktop.org 桌面提示支援
正簡轉換
自動英文模式
標點支援

外觀

字型：AR PL UMing TW Bold (12)
選單字型：FreeMono (16)
要顯示的字形語系：zhTW_.UTF-8
狀態面板隱藏模式：自動
