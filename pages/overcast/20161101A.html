適合電腦桌面的 Linux 發行版

這是我為 2022 年 Linux 發行版所做的推薦排名。

排名的基準是電腦桌面！若已經熟悉 Linux，打算將它安裝在家用多媒體電腦或商用文書機，就適合參考這份排名。

若是《適合入門 Linux 的發行版》《適合雲端的 Linux 發行版》《適合終端的 Linux 發行版》，會有很不一樣的排名。

因此本排名並不是對 Linux 發行版做高下之分，優劣與否，隨用途（軟體）和條件（硬體）取捨而定，沒有真正的誰優誰劣～

TOP 1: Ubuntu MATE

可以取代輕量級 Lunbuntu 和入門級 elementary OS 的 Linux 發行版！而且跟 Linux Mint 一樣耐操～

Lubuntu 18.10 從 LXDE 改為 LXQt 後，變成所有輕量級 Linux 發行版中最笨重的！甚至 Lubuntu 20.04 開始，不再提供 32 位元電腦的版本，已不是老舊電腦能用的 Linux 發行版，建議改用 Ubuntu MATE，這是 Raspberry Pi 推薦的第三方作業系統，可見其輕量。

在 elementary OS 創辦人 Cassidy James 被逼退，原先員工也全被遣散下，這意味著它開始變質了，不建議使用，建議改用 Ubuntu MATE，它提供 Windows、macOS、Gnome…多種桌面環境可以切換，簡單好入門。

Linux Mint 使用基於 GNOME 改造的桌面環境：MATE 和 Cinnamon，這是最耐操桌面環境！缺點是系統需求偏高，電腦老舊一點就跑不太順，所以建議改用 Ubuntu MATE。桌面工作最重要的，就是耐操：「久操不卡、亂按不壞。」若沒有專業人士需要量身打造工作環境的特殊需求，選 Ubuntu MATE 就對了！

TOP 2: Pop!_OS

如果你嫌 Ubuntu MATE 操作環境像舊時代的作業系統，配不上你買的先進電腦，那就改用 System76 品牌電腦精心打磨的 Pop!_OS。

Pop!_OS 有現代化的互動式操作體驗，回饋感十足、美觀酷炫、卻又無比流暢～

任何 Linux 發行版，都有被嫌棄的缺點，但我還沒聽過有人嫌 Pop!_OS 不好的，頂多嫌它基於 Ubuntu。也確實 Ubuntu 越來越商業考量，明明有公認更好、更開放的方案，卻總是獨創自家專屬規格想寡佔市場，已不再是 Mark Shuttleworth 掏錢創立時「群在故我在」那廣受讚譽的風評。

話題扯遠了，總之呢，從其它 Linux 發行版改用 Pop!_OS 的人，都會說再也回不去了！這種感覺就像 1999 年時，從 Red Hat Linux 改用 Linux-Mandrake 後就回不去了一樣！

TOP 3: SteamOS

專為全球第一大遊戲訂閱平台 Steam 量身打造的 Linux 作業系統！要玩電腦遊戲的話，非它莫屬！

SteamOS 致力於發揮硬體性能，以及與 Windows 軟體的相容性。

然而，開發商 Valve 著重的是遊戲選單的操作介面，而不是桌面工作環境，所以在桌面的表現只是半吊子，它更成熟的體驗是類似 PlayStation 和 Xbox 登入帳戶後的畫面那樣，展現 Steam 平台的內容，讓玩家能直覺地獲取遊戲。雖然致力於 Windows 軟體的相容性，卻不重視 Linux 程式的可用性，能方便安裝在 SteamOS 使用的應用軟體不多～

純粹以其絕無僅有的特殊性排第三，若不玩遊戲，SteamOS 反而是最差的 Linux 發行版，不建議電腦工作者使用…也不是不能使用，但其它 Linux 發行版做得更好。

TOP 4: Manjaro

在 Arch Linux 極簡、高效的系統基礎上，致力於為它打造人性化的使用環境。

桌面環境預設使用 Xfce，有自己的軟體庫，這發行版的情況，很像基於 Debian 的 Ubuntu，在完美中再進一步苛求，所以我想這發行版終將表現出如同 Ubuntu 般的影響力，為 Linux 發行版帶來新的境地。

然而訴諸的是為不夠人性化的 Arch Linux 提供人性化，讓更多人能受用 Arch Linux 的高效系統表現，並不是桌面工作的使用環境，因此電腦桌面這部份，沒有基於 Ubuntu 的 Linux 發行版來得用心，好像有納入桌面環境和檔案管理器便了事，不會進一步整合、優化。

Manjaro 比較適合專業工作者使用，像是經常測試程式的軟體工程師、或需要耗用大量系統資源的影像處理人員。若覺得專為桌面環境打造 Linux 發行版大多效能低落、無法滿足你的需求，往往可以在 Manjaro 得到滿足，因此排第四。

TOP 5: Fedora Linux

原 Red Hat Linux 的發行版，目的並不是做為桌面工作的作業系統，而是系統功能卓越的作業系統，工作上比較符合老派 UNIX 的應用精神，因此獲得 Linus Torvalds 認可，並使用這個發行版。

以 Linux 發行版來說，這是比上述發行版更卓越的作業系統！適合架站、佈署主從架構、或當 UNIX 工作站使用。但對桌面工作來說，東西太多，塞了一堆桌面工作者用不到的東西，有點大材小用了！雖然也不是說不能當桌面工作的作業系統，但這比較像電腦工程師用的桌面作業系統，因此排在第五。

附帶一提，還有規格等同 Red Hat Enterprise Linux 但免費開放使用的 CentOS Stream，不過除非真的需要架一個企業級的伺服器，不然就像 Windows Server 2016 一樣，桌面工作者還是用 Windows 10 就好～

並不是最強的作業系統就一定好用，適不適合才是好用的關鍵。換句話說也不是視窗桌面就一定好用，有些工作場合像是收銀台或機電整合，把操作寫死在一目了然的簡陋介面或用指令下動作反而好用，在層層收納的圖形界面翻來找去只是整死自己。
