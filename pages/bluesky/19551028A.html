我可不喜歡比爾蓋茲這個人

看比爾蓋茲現在做慈善，裝好人，真覺得噁心！就我從 Windows 95 到 Windows XP 體驗過的糟糕往事，他根本就不慈善、也不是好人。

他經營微軟時，挾 Windows 市佔率綁自家三流軟體鬥垮對手的產品（例如 Netscape Navigator）、自訂格式破壞標準規範故意製造不相容藉以排擠對手（例如不甩 W3C 的 HTML）？如果這麼做是為了讓使用者有更好的體驗那就算了，但微軟這麼做往往造成使用者只能使用體驗很糟的軟體。他只是為了讓公司稱霸業界、主宰一切而已，才不管使用者和同業公司怎樣！

不但用卑劣的商業行為剷除對手，還讓客戶只能使用三流的軟體，根本就是製造雙輸的生意人，只有他自己一個人贏而已。這種人現在做慈善、裝好人，真是噁心！

當初經營微軟時，怎麼就不慈善事業一點共創雙贏？怎麼就不好人好事代表一點只為客戶著想？反而只會塞自家不怎樣的產品像餵狗吃屎一樣硬要使用者吞下去，然後把真正一流的產品鬥垮沒得用。我看他不是為了造福別人做慈善，而是為了自己名聲和形象做慈善吧？
