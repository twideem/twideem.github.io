Windows 免費軟體清單

Windows 給人的刻板印象就是 Money，跟 Windows 有關的就是錢；而錢會在許多人心中刻下深深的印象難以磨滅的原因則是另外一個字眼：「昂貴」，常常讓一般人用不起。[1]

雖然我自己也是 Freeware (免費軟體) 的使用者，但我覺得如果是因為在網路看了這些批評微軟 Windows 的文章，而讓你也跟著敵視 Windows 的話，其實蠻可惜的。因為，可安裝在 Windows 的免費軟體跟 Linux 數量差不多，而且拜 Windows 方便的桌面環境所賜，這些免費軟體其操作介面比 Linux 更好用而且穩定。

一般學生常用到的軟體大概是 Microsoft Office、PhotoShop、CorelDRAW、AutoCAD、3DS MAX、Dreamweaver、譯典通、Visual Basic、UltraEdit、IIS，而這些軟體都可以在網路上找到免費的替代方案。

本文列出個人所整理的免費軟體清單，相信能讓你體會到，Windows 也能夠以最低的成本、整合眾多免費軟體搭建出穩定的開發環境！

不過，本軟體清單對使用者來說有個地方蠻缺憾的，就是沒有刻意挑選中文的語系，使用上有很大隔閡。另外，這份清單的在軟體歸類方面，有個需要解釋的地方，就是中文字型歸類在資料管理，英文字型卻歸類在影音繪圖；這是因為在台灣使用到英文字型的場合，通常是以平面設計的用途居多，因此將英文字型歸類在影音繪圖裡面，而中文字型多應用在文書處理的場合，故歸類在資料管理。

使用前警告

本文章推薦的應用軟體，有可能造成您硬體設備的損毀與軟體系統的破壞，並連帶產生身心上的傷害。對此，本人不願意承擔您操作上所造成之任何物質上的損失，也不願意賠償精神損害。若您無法自行承擔不良後果，請勿下載、安裝、使用本文章推薦的應用軟體。

而當您打算使用這些應用軟體，請事前洽詢您購買電腦的經銷商，了解本單元內容極有可能造成無法挽救的設備損害，並事先找好身心科醫師，以便發生損害時能心理諮商您受創的身心。

本單元所介紹的免費軟體，往往會置入性行銷一些廣告軟體，因此安裝時請勿無腦狂按下一步，必須仔細看清楚每個安裝步驟，取消你不想要的設定！不要輕忽這個問題，有些外掛被置入後，很難移除乾淨！要命的是，不想安裝置入性行銷軟體的設定方式越來越複雜，早先是取消勾選安裝項目，後來出現要按 Decline 鈕才表示不安裝，取消勾選的只是版權條例，你以為取消勾選了，按下 Next 後變成安裝。甚至還有綁了好幾款廣告軟體，每種軟體的取消安裝方式都不一樣，再怎麼小心，我們依然很容易誤入陷阱把不想的東西給安裝上去。

因此，安裝本文所介紹的免費軟體，是很大風險的行為。只要有綁置入性行銷廣告軟體，你就非常有可能安裝到電腦裡面，安裝以後有可能再也無法解除安裝。因此，再強調一次！若你無法自行承擔安裝惡意廣告軟體破壞你 Windows 系統的風險，請勿照本文的介紹去安裝軟體！因為你的損失，本人概不負責！

選用付費軟體的優勢

Windows 跟 Linux 一樣有很多免費軟體可以滿足使用上的需求，而 Windows 更是「只要花錢就有更好的軟體可用」。雖然你可能還是對花錢買軟體有點反感，但我認為這反而是一項優勢，最好的例子就是 Dreamweaver 與 Flash，若是網頁設計工作者，我寧願在 Windows 花幾萬塊買下這些軟體，也不會比在 Linux 沒有 Dreamweaver 和 Flash 可用來得吃虧！

這裡想強調一個想法就是，除了厭惡昂貴的商用付費軟體，不妨轉而思索看看花這些錢能為你提高多少工作效益，這才是使用軟體的重點。有時候花錢更能讓你在電腦工作上佔得優勢，若讓反感的心態讓你錯失了這樣的機會反而可惜。

接著，很多需要付費的商業軟體 (或直接稱商品)，價格實在很貴，看了就讓人想要破口大罵；這種觀念必須糾正過來。老實說，很多商業軟體本來就不是讓「你」使用的，而是讓「企業」使用的。市面上所見的商業軟體，其價位大致分為「一般民眾」「專業人士」「企業用戶」三種階級，而這些商品根據這三種階級所開設的價位並未誇張到哪裡去。比如說，針對一般民眾所設計的影像處理軟體 PhotoImpact，它的售價是 NT$ 3,900，請問有貴得很誇張嗎？而針對專業繪圖人員所開發出來、做為其生產工具的 Photoshop，售價大約 NT$ 24,000，吃飯工具花一個月薪水來買有很要命嗎？至於有些學生囔著太貴的 Windows Server 和 SQL Server 搭配起來更是數十萬跑不掉，但這套系統當初就不是設計給學生與一般民眾用的，而是設計給企業戶營運用的，在你愛用而咒罵微軟死要錢的同時，先想想是不是自己理虧？

也有不少人對於花個幾千塊買顯示卡覺得理所當然、甚至划算，但對於許多一千元有找的優質軟體像是自然輸入法、WinRAR…聽到有人花錢去買就暗地笑他傻子，好像硬體這樣看得到的東西才有價值，軟體這種無形東西就沒有成本可言似的。

整理這份免費軟體清單的目的，不是為了反販售而使用免費軟體！讓免費軟體相容付費軟體，就能在應有盡有的情況下，重點提升更高的人工作業效率，這點是我認為最有價值的方案。

本站整理的免費軟體清單以精選為原則，同樣的軟體就只挑選最順眼的一個來上架。這樣容易因為我的不專業，挑選出不甚理想的軟體。因此下列網站整理出來的清單或許才是你想要的，建議你瀏覽看看：
