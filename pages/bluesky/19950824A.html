修改註冊的使用者

這是在安裝 Windows 時填寫的使用者資訊，有「姓名」和「公司」兩筆。

Windows 並未直接提供修改這兩筆資料的程式，必須使用「登錄編輯程式」修改。

Windows 95/98/ME

按「開始」→「執行」→「regedit」，開啟「登錄編輯程式」。

切換到 HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion。

在 RegisteredOrganization 可修改公司，在 RegisteredOwner 可修改姓名。

Windows XP/Vista/7/8/10

跟 Windows 9x 一樣，只是切換機碼時 Windows 改成 Windows NT 而已：

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion
