Windows XP 支援終止的解決方針

2014 年 4 月 8 日起 Microsoft 將不再提供 Windows XP 售後的技術服務！但這不表示你不能繼續用它來打報告或玩遊戲，只是不應該再用它來上網而已，否則有系統漏洞沒辦法補的話，你的電腦會淪為不肖駭客違法亂紀的工具。

辦公電腦、商用電腦

大多數上班用的電腦其實不需要上網，因為是去辦公的，不是去網路聊天的。因此辦公室裝 Windows XP 的電腦，建議公司把網路線拔掉，即可繼續用下去～

公司必須透過電子郵件或即時傳訊來聯繫事務的話，雖然只好升級作業系統，但因為當初裝載 Windows XP 的電腦應該跑不動 Windows 7，不如整台換掉。

家用電腦

不需要上網的話，可以拔掉網路線，當作 Windows 舊遊戲專用機繼續用。

既然如此，許多安全性更新都是多餘的，因此不妨重灌 Windows XP，記得不要連接網路，而是用打電話的方式啟用產品，然後用離線安裝版的 WindowsXP-KB936929-SP3-x86-CHT.exe 升級為 Service Pack 3，再用離線安裝版的 directx_Jun2010_redist.exe 升級為 DirectX 9.0c，讓這台 Windows XP 遊戲專用機有更乾淨而俐落的執行性能。

需要上網的話，建議安裝免費的 Lubuntu 作業系統來上網，它不只能讓舊電腦跑得比 Windows XP 還順，還會建立多重開機選單，讓你可以回 Windows XP 玩遊戲。
