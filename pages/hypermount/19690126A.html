陣列

如果建立的是一連串同類型的資料，不妨使用「陣列」來完成。

產生陣列

陣列的宣告方式有多種，底下依推薦的順序列出，挑你自己習慣的寫法即可，不見得要混著用。

標準的陣列宣告方式

var 陣列名稱 = new Array(資料一, 資料二,…);

簡便的陣列宣告方式

var 陣列名稱 = [資料一, 資料二,…];

宣告不含資料與筆數的空陣列

var 陣列名稱 = new Array();

初始化資料筆數的陣列（並不是限制資料筆數）

var 陣列名稱 = new Array(初始化陣列筆數);

二維陣列

var 陣列名稱 = new Array();
陣列名稱[索引值] = new Array();

使用陣列

底下是存取陣列的做法。

存入資料

陣列名稱[索引值] = 資料;

取出資料

陣列名稱[索引值];

索引值由 0 開始，而不是 1。

換句話說，陣列的第一筆資料是 [0]，不是 [1]。

想當然，最後一筆資料就是 [資料筆數-1]，通常我們會用陣列的 length 屬性來取得資料的筆數。

陣列的進階應用

由於 JavaScript 的陣列是 Array 物件，因此陣列可以呼叫許多物件功能，來完成很多進階的應用。

於首端加入資料

1) 陣列名稱.unshift(資料);
2) 陣列名稱.unshift(資料1, 資料2,…);

於尾端加入資料

1) 陣列名稱.push(資料);
2) 陣列名稱.push(資料1, 資料2,…);

插入資料

1) 陣列名稱.splice(第幾筆資料開始, 0, 插入的資料);
2) 陣列名稱.splice(第幾筆資料開始, 0, 插入的資料1, 插入的資料2,…);

取代資料

1) 陣列名稱.splice(第幾筆資料開始, 1, 插入的資料);
2) 陣列名稱.splice(第幾筆資料開始, 被取代的資料筆數, 插入的資料1, 插入的資料2,…);

擷取資料（節取出來會產生新的陣列）

陣列名稱.slice(第幾筆資料開始, 第幾筆資料結束);

結合進其它陣列

陣列名稱.concat(另一個陣列名稱);

移除第一筆資料

陣列名稱.shift();

移除最後筆資料

陣列名稱.pop();

移除指定範圍的資料

陣列名稱.splice(從第幾筆資料開始, 需要移除的資料筆數);

排序資料

陣列名稱.sort();

反轉資料（將資料順序顛倒過來排列，並不是逆向排序。）

陣列名稱.reverse();

搜尋資料

1) 陣列名稱.indexOf(資料);
2) 陣列名稱.indexOf(資料, 從第幾筆開始搜尋);

以分隔符號將陣列資料輸出為字串

陣列名稱.join(分隔符號);
