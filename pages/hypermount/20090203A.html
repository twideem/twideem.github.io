如何使用 SyntaxHighlighter 顯示程式範例

SyntaxHighlighter 是由 Alex Gorbatchev 開發，能夠在網頁中，用高亮度識別文字，來顯示程式碼範例的 JavaScript 系統。


Alex Gorbatchev 這套 SyntaxHighlighter 系統，不僅預設的功能即符合大眾需求，往後也可以透過簡單的參數設定來調整功能，顯示出來的結果清爽又美觀，成為這類系統中最受歡迎的一款。

請從下列官網的網址下載：

http://github.com/syntaxhighlighter/

改變高亮度主題

shCoreDefault.css 這個檔案就是預設的高亮度主題樣式，你可以改用以下檔案，來嘗試各種不同的高亮度樣式：

shCoreDjango.css
shCoreEclipse.css
shCoreEmacs.css
shCoreFadeToGrey.css
shCoreMDUltra.css
shCoreMidnight.css
shCoreRDark.css
補充一件事，其實 shCoreDefault.css 是 shCore.css 與 shThemeDefault.css 合併起來的檔案，如果希望常常切換主題樣式的話，可以先載入共通設定的 shCore.css，再載入其它不同的 shThemeXXX.css 來套用。

如果你希望所有主題的字型與大小都是統一的，其它 shThemeXXX.css 只是變換顏色而已，就適合用 shCore.css + shThemeXXX.css 的方式。反過來，希望每個主題的字體、位置、顏色都不一樣的話，就適合用 shCoreXXX.css 的方式。

進階設定

改用其它標籤來顯示程式碼

預設只能用 pre 標籤來高亮度識別程式碼，如果你想改用其它標籤的話，可以在 JavaScript 執行如下程式：

SyntaxHighlighter.config.tagName = 'code';

關閉左邊的行號

在高亮度顯示用標籤的 class 屬性裡，加入 gutter:false 即可。

關掉右上角的 ? 方塊

在高亮度顯示用標籤的 class 屬性裡，加入 toolbar:false 即可。

如果希望預設全部關掉的話，則用 JavaScript 來執行這段程式：

SyntaxHighlighter.defaults.toolbar = false;

折疊起來

在高亮度顯示用標籤的 class 屬性裡，加入 collapse:true 即可，但 toolbar 必須為 true，否則會整個不見。

其它輔助撰寫的設定

高亮度顯示用標籤的 class 屬性中，還有一些方便我們書寫的格式化設定項目，像是：

html-script:true

可以讓 < > 避免被視為 HTML 標籤而無法顯示，但記得在 brush:xml 時無效。

tab-size:2
用來指定 Tab 的大小，上例設定為 2 個空白。

stripBrs = true

可以讓程式碼範例裡面的 <br> 不顯示出來。

還有更多設定項目，請參考官網的 Configuration 文件。

[*] 你可以透過《線上切換 SyntaxHighlighter 樣式》測試各種樣貌。
