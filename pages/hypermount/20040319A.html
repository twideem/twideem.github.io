在網頁瀏覽器將 Markdown 轉為 HTML

下載 Showdownjs

http://showdownjs.com

使用方法

將下載的壓縮檔解開，在 HTML 載入 dist 資料夾裡的 showdown.min.js。

然後寫 JavaScript 程式，用 new showdown.Converter() 產生 Showdownjs 物件，再對該物件操作 makeHtml() 將 Markdown 格式的文字轉換為 HTML 格式的內容。

最後將 HTML 格式的內容指定給負責顯示結果的標籤即可。

用法示範

底下範例將 Markdown 格式的文字寫在 HTML 的 P 標籤，然後用 Showdownjs 將它轉成 HTML 格式的內容後，直接覆寫到 P 標籤：

顯示結果
