jQuery 速成指南

安裝

只要用 <script></script> 載入 jQuery.js 檔案，就可以編寫 jQuery 程式。

直接載入網路的 jQuery 檔案

<script src='http://code.jquery.com/jquery.min.js'></script>

至官方下載 jQuery 檔案

http://jquery.com

如果程式執行於保持連線的使用環境，建議直接載入官網的 jQuery.js，而且隨時都是最新版本。

如果程式執行於離線環境，可以下載 jquery.js，然後與你的專案放在一起使用。

下載時有分兩種版本：壓縮精簡過的 jquery.min.js，以及未經壓縮的 jquery.js。前者檔案小，如果你的程式放在網路上來使用，建議載入這個版本。後者檔案大，但可讀性高，講究程式碼要清清楚楚、方便閱讀以調查問題所在的人，可以考慮使用。

選擇器基本用法

要說 jQuery 的重頭戲，就是可以快速、方便地，指定網頁裡面你想要的元素來使用，這稱為「選擇器」(Selector)。

選擇標籤元素

jQuery('標籤')

用標籤的 ID 名稱選擇元素

jQuery('#識別名稱')

用標籤的 CLASS 類型選擇元素

jQuery('.類型')

選擇器進階條件

jQuery 可以針對選擇出來的結果進行條件匹配，進一步篩選出更精確的元素。

依順序限定條件

:eq(索引值)依陣列索引值指定第幾個元素
:first第一個元素
:last最後一個元素
:lt(個數)最前幾個元素
:gt(個數)最後幾個元素
:odd選出奇數順序的元素
:even選出偶數順序的元素

依內容限定條件

:has(子標籤)擁有子標籤的元素
:contains(文字)含有文字的元素
:empty內容為空的元素

依狀態限定條件

:enabledenabled（啟用）狀態的元素
:disableddisabled（禁用）狀態的元素
:selectedselected（選取）狀態的元素
:checkedchecked（核取）狀態的元素
:visiblevisible（顯現）狀態的元素
:hiddenhidden（隱藏）狀態的元素
:animated正在執行動畫狀態的元素

依元素限定條件

:headerH1 到 H6 等元素
:inputINPUT 元素
:textTEXT 元素
:passwordPASSWORD 元素
:radioRADIO 元素
:checkboxCOECKBOX 元素
:submitSUBMIT 元素
:resetRESET BUTTON 元素
:buttonBUTTON 元素
:imageIMG 元素
:fileFILE 元素

其它限定運算式

jQuery(this)當前的元素
jQuery('標籤>子標籤')含有子標籤的標籤
jQuery(條件一, 條件二,…)建立多個條件
:not(條件)不符合選擇器條件的元素

項目繁多，所以稍微舉幾個例子做為示範，其它功能請依照樣式自行嘗試：

jQuery('span:first')選擇網頁中的第一個 span 元素。
jQuery('span:eq(1)')選擇網頁中的第二個 span 元素。
jQuery('not(span:eq(0))')選擇網頁中第一個 span 元素以外的其它元素。
jQuery(:radio)選擇網頁中所有 RADIO 元素。
jQuery(:radio, :checkbox)選擇網頁中所有 RADIO 元素與 CHECKBOX 元素。
jQuery('header>h2')選擇含有 H2 的的 HEADER 元素。
