Firefox 設定集

使用者設定檔

Windows 8

「螢幕右角」→「搜尋」→「%APPDATA%\Mozilla\Firefox\Profiles\」→備份或覆蓋「xxxxxxxx.default」這個資料夾。

關閉九宮式新分頁

Firefox 40 為止的做法

1)「擴充套件」→「Firefox 首頁」→「移除或停用」→「立刻重新啟動」
2)「網址列」→輸入「about:config」
3)「browser.newtab.url」→「網址或 about:robots（空白頁）」
4)「browser.newtabpage.enabled」→「false」

Firefox 41 以後的做法

因為 Mozilla 無法解決駭客綁架首頁的問題，所以從 Firefox 41 開始，取消了 browser.newtab.url，只好下載擴充套件 New Tab Override：

http://addons.mozilla.org/zh-TW/firefox/addon/new-tab-override/

或只能開啟為首頁的 New Tab Homepage：

http://addons.mozilla.org/zh-TW/firefox/addon/new-tab-homepage/

Firefox 57 以後的規則

因為 WebExtensions 的規範，附加元件無法將新分頁導向本機網頁，只能讓新分頁設定為特定網站了～

取消意外關閉 Firefox 時自動恢復瀏覽頁的功能

在「網址列」輸入 about:config，將 browser.sessionstore.resume_from_crash 設為 false。

關閉下載掃毒

在「網址列」輸入 about:config，將 browser.download.manager.scanWhenDone 設為 false。

不要再詢問通知

在「網址列」輸入 about:config，將以下項目設為 false：

dom.push.connection.enabled
dom.push.enabled
dom.webnotifications.enabled
dom.webnotifications.serviceworker.enabled

不要再詢問透露地理位置

在「網址列」輸入 about:config，將 geo.enabled 設為 false。

Firefox 已在執行中，但無回應。

如果設定「選項」→「個人隱私」→「歷史紀錄」→「使用自訂的設定」的話，關閉 Firefox 後想再執行時，經常會出現：「Firefox 已在執行中，但無回應。請先結束現有的 Firefox 或重新開機再開啟新視窗。」每次都要叫出「工作管理員」結束 Firefox 才能如願執行，相當惱人！

改回「保留所有歷史紀錄」可以解決問題，或者自己寫個「先砍 firefox.exe 程序，再執行 firefox.exe 的批次檔」做為啟動 Firefox 的程式：

taskkill /F /IM firefox.exe
'C:\\Program Files (x86)\Mozilla Firefox\firefox.exe'
