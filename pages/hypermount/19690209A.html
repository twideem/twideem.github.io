判斷

依據條件，判斷程式應該執行哪一段流程。

if else

判斷條件式是否成立，然後執行程式。

單一條件式

if(條件式) 程式敘述

多重條件式

if(條件式) 程式敘述一
else 程式敘述二

如果條件式經由布林運算結果為 true，則執行程式敘述一，否則執行程式敘述二。

程式敘述不只一行的話，必須用 {} 將一連串的程式敘述形成區塊：

if(條件式){
　程式敘述１;
　程式敘述２;
　程式敘述３;
}
此外，else 後面的程式敘述二，可以是另一個 if 條件式，形成更多重的條件判斷式：

if(條件式一) 程式敘述一
else if(條件式二) 程式敘述二
else if(條件式三) 程式敘述三
else 程式敘述四

switch case

根據條件的計算結果，執行對應的程式。

決策判斷式

switch(條件){
　case 運算結果一:
　　程式敘述一;
　　break;
　case 運算結果二:
　　程式敘述二;
　　break;
　default:
　　程式敘述三;
}

當「條件」為「運算結果一」，就執行「程式敘述一」，為「運算結果二」，就執行「程式敘述二」，都不符合的話，就執行「程式敘述三」。

其中 default 不需要的話可以省略，而每個 case 會用 break 指令來結束整個 switch 程式，否則會繼續執行下一個 case。不過如果 switch 最後一個是 case，就沒有這個問題，可以省略。

接著舉個範例如下，假設有個保存所通關密語的 password 變數，如果是 GOLD 就開放黃金會員權限，如果是 SILVER 就開放白銀會員權限，如果是 IRON 則開放普通會員權限……
